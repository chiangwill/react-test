# 這裡是自救會ㄇ？

1. 可能要先安裝一下 nvm (一種版本管理器)
    - [nvm for windows](https://github.com/coreybutler/nvm-windows)

    - MacOS
        ```
        $ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
        ```

2. 接著用 nvm 安裝 node

    ```
    $ nvm install 16
    ```

    然後試試看下述指令有沒有返回 node 版本
    ```
    $ node -v
    ```
